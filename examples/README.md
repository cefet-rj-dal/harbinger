# Harbinger Examples

This directory contains the source R Markdown notebooks used to generate the material under `/examples/`. The goal of the collection is to help the reader learn the package in a sensible order instead of facing only a long list of methods.

If you are new to `harbinger`, start with the tutorials, then use the other folders as focused reference tracks. The links below point to the rendered Markdown files that are intended for reading.

- [Tutorials](/examples/tutorial/README.md) - 10 notebooks organized as a guided path from first contact with the workflow to motifs and symbolic analysis.
- [General](/examples/general/README.md) - package orientation, utility helpers, and evaluation objects.
- [Data](/examples/data/README.md) - benchmark collections and domain datasets used across the examples.
- [Transformations](/examples/transformations/README.md) - smoothing and symbolic encodings that prepare a series for later analysis.
- [Anomalies](/examples/anomalies/README.md) - anomaly detection methods from simple baselines to ensembles, autoencoders, and multivariate analysis.
- [Change Points](/examples/change_point/README.md) - change-point methods from single-break intuition to structural and volatility-oriented approaches.
- [Motifs](/examples/motifs/README.md) - repeated-pattern and discord discovery with Matrix Profile and symbolic approaches.
- [Custom](/examples/custom/README.md) - extension examples for readers who want to plug their own components into the framework.
