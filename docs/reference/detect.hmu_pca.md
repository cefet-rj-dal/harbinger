# Multivariate anomaly detector using PCA

Multivariate anomaly detector using PCA

## Usage

``` r
# S3 method for hmu_pca
detect(obj, serie, ...)
```

## Arguments

- obj:

  detector

- serie:

  time series

- ...:

  optional arguments.

## Value

A dataframe with information about the detected anomalous points

## Examples

``` r
detector <- harbinger()
```
