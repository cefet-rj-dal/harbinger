# Time Series Adaptive Normalization (Exponential Moving Average - EMA)

Time Series Adaptive Normalization (Exponential Moving Average - EMA)

## Usage

``` r
ts_ean(remove_outliers = TRUE, nw = 0)
```

## Arguments

- remove_outliers:

  logical: if TRUE (default) outliers will be removed.

## Value

a `ts_ean` object.
