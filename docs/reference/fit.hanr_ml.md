# Fits a detection method using a regression model

Takes as input a "Harbinger" object and a time series

## Usage

``` r
# S3 method for hanr_ml
fit(obj, serie, ...)
```

## Arguments

- obj:

  detector

- serie:

  time series

- ...:

  optional arguments.

## Value

Adjusted model

## Examples

``` r
detector <- harbinger()
```
