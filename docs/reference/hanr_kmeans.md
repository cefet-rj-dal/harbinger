# Anomaly detector using kmeans

Anomaly detector using kmeans

## Usage

``` r
hanr_kmeans(seq = 1, centers = NA, alpha = 1.5)
```

## Arguments

- seq:

  Sequence size

- centers:

  Number of centroids

- alpha:

  Threshold for outliers

## Value

hanr_kmeans object

## Examples

``` r
detector <- harbinger()
```
