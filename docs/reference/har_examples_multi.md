# Synthetic multivariate data set for time series analysis Each time series has 101 observations with a single event on it.

Synthetic multivariate data set for time series analysis Each time
series has 101 observations with a single event on it.

## Usage

``` r
data(examples_anomalies_multi)
```

## Format

A list of multivariate time series.

## Source

[Harbinger package](https://github.com/cefet-rj-dal/harbinger)

## References

Salles et al. (2020) SBBD p. 73-84 ([Harbinger
Framework](https://doi.org/10.5753/sbbd.2020.13626))

## Examples

``` r
data(examples_anomalies_multi)
serie <- examples_anomalies_multi[[1]]
```
