# Fit a general detector

Takes as input a "Harbinger" object and a time series

## Usage

``` r
# S3 method for harbinger
fit(obj, serie, ...)
```

## Arguments

- obj:

  detector

- serie:

  time series

- ...:

  optional arguments.

## Value

The "Harbinger" object without doing any processing or tuning of the
model

## Examples

``` r
detector <- harbinger()
```
